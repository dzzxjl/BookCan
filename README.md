### BookCan(WiKi)
书要系统地读才能真正理解书，
BookCan中文即书罐。
将以书罐的形式将相应门类知识与具体书籍挂钩，
加深读者的印象、理解，构建知识体系。

BookCan的slogan 排列组合的书目
* BookCan以书罐的形式将具体书籍与所属知识门类挂钩，帮助你快速发现书籍，收获系统知识，享受阅读乐趣
